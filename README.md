# ETL Pipeline for Sales Data Warehouse and Analytics

## 📌 Project Overview

This project demonstrates a complete ETL (Extract, Transform, Load) pipeline for sales data using Python. The pipeline extracts raw sales data, cleans and transforms it, loads it into a SQLite data warehouse, and visualizes business insights using Power BI.

---

## 🎯 Objectives

- Extract sales data from CSV
- Clean and transform data
- Remove duplicates
- Handle missing values
- Convert data types
- Load cleaned data into SQLite
- Generate ETL reports
- Create an interactive Power BI dashboard

---

- Python

## 📂 Dataset

- Source: Kaggle Superstore Sales Dataset
